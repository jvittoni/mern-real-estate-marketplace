# Real Estate Marketplace Web App

## Description
The Real Estate Marketplace Web App is a full-stack application built using the **MERN stack** and **Tailwind CSS**. It enables users to browse, search, and manage real estate listings with a responsive and intuitive interface. Advanced authentication and user-friendly features provide a seamless experience for both buyers and sellers.

<br>

## Features
- **View & Search Listings:** Users can browse and filter real estate listings efficiently.
- **User Authentication:** Sign in via email/password or Google OAuth with secure JWT-based sessions.
- **Create & Manage Listings:** Users can create, edit, and delete their own property listings.
- **Image Uploads:** Add property images to listings for better visualization.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Advanced Search:** Filter properties based on location, price, and other criteria.

<br>

## Frontend
- **Framework:** React 19 with Vite
- **Routing:** React Router DOM
- **State Management:** Redux Toolkit with React-Redux and Redux Persist
- **Styling:** Tailwind CSS
- **UI Components:** React Icons, Swiper for carousel functionality
- **Authentication:** Firebase for authentication management
- **Development Tools:** ESLint, Vite Plugin React SWC, Type Definitions for React, Tailwind Line Clamp
  
<br>

## Backend
- **Framework:** Express.js 4
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT (jsonwebtoken) and bcryptjs for password hashing
- **Cookies:** Cookie-parser for session management
- **Environment Variables:** dotenv
- **Development Tool:** Nodemon for live server reloading
